# Listado de tareas

Primero, para iniciar el servidor debes ejecutar el comando:

```bash
npm run dev
# o
yarn dev
```

## Incidentes

- [ ] La imagen de cada noticia luce desproporcionada.
- [ ] Al buscar por el titular de la noticia, los resultados no son los correctos.
- [ ] Al realizar una búsqueda la imagen de la noticia no es la correcta.

## Mejoras

- [ ] Abrir la noticia en una nueva pestaña.
- [ ] Filtrar las noticias por el titular o la descripción.
- [ ] Realizar la búsqueda después de 3 segundos de haber dejado de escribir.
